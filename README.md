## About The Project

More informations can be found at this notion page: https://www.notion.so/kanastra/Hiring-Challenge-Soft-Engineers-Fullstack-Live-or-Take-home-Version-02bd3892c6b946e1b0b801f64638fb25?pvs=4

Get yourself acquaint with the test before diving into the solution by reading this preparation note: https://www.notion.so/kanastra/Prepara-o-544501f41a5b422e86052efdb8e0ae94?pvs=4

### Built With

* [React](https://reactjs.org/)
* [Typescript](https://www.typescriptlang.org/)
* [Tailwindcss](https://tailwindcss.com/)
* [Vite](https://vitejs.dev)
* [shadcn/ui](https://ui.shadcn.com/)
* [bun](https://bun.sh/)

In the backend you can choose either to go with Python (django) or PHP (laravel).
